# 01 · 模型层 · MODEL LAYER

纯语言模型，**无状态**。接收对话历史 + 工具定义，返回文本或工具调用请求。
不知道终端、文件系统为何物。

## 职责
- 输入：对话历史（messages）+ 可用工具的 schema 定义
- 输出：文本回复，或一个 / 多个工具调用请求
- 不执行任何副作用，纯决策

## 与其他层的关系
- 被 [`agent`](../agent/) 调用，结果回填后再次被调用
- 可用工具来自 [`tools`](../tools/)（含 [`extension/mcp`](../extension/mcp/) 注册的外部工具）
- 行为由 [`prompt`](../prompt/) 用自然语言约束
