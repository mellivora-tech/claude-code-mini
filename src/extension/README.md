# 横切层 · 扩展 / 注册层 · EXTENSION LAYER

不是新增的「第八层」，而是贯穿全栈的设计原则：核心层（工具、知识）天生可插拔。
运行时先注册一个轻量「目录条目」，真正内容延迟到被需要时才加载 —— 避免一次性塞爆上下文。

## 两个插槽

| 机制 | 挂载点 | 说明 |
| --- | --- | --- |
| [`mcp/`](mcp/) | 第 02 层 工具层 | 外部 server 上报工具 schema，注册进工具列表，编排层统一调度并转发执行，产生真实副作用。 |
| [`skill/`](skill/) | 第 06 层 提示层 | SKILL.md 上报简短索引，模型按需触发读取，内容并入上下文，纯信息无副作用。 |
