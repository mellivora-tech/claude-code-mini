# 记忆 · 三种不同的时间尺度

「记忆」不是单一机制，而是三套独立系统，生命周期、触发方式、存储位置都不同。

| 尺度 | 目录 | 生命周期 | 存储 |
| --- | --- | --- | --- |
| 一次对话 | [`session/`](session/) | 对话一关全部消失 | 上下文窗口，无持久化 |
| 单个项目，强制加载 | [`project/`](project/) | 每次会话启动强制全量读入 | 文件系统（CLAUDE.md） |
| 跨项目，按需检索 | [`cross-session/`](cross-session/) | 跨会话长期保留 | 独立数据库（RAG） |
